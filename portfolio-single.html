프로젝트 명 : KH-CAFE ERP

커피의 재료인 원두, 설탕 등 원자재의 발주부터 입고, 출고 등 재고관리와 주문 관리, 통계 등을 하나의 프로그램으로 통합 관리할 수 있게 하여 직원들의 업무 효율성을 증가시킨다.

구현 기능

회원서비스 : 로그인, 로그아웃, 회원가입, 이메일 인증, 스프링 시큐리티, ID&PW 찾기

테이블 : 상품관리, 발주관리, 입고관리, 출고관리, 주문관리, 직원관리

통계 : 통합매출관리(일별, 주별, 월별)

엑셀 업로드 : 상품관리, 발주관리, 입고관리, 출고관리, 주문관리

엑셀 다운로드 : 상품관리, 발주관리, 입고관리, 출고관리, 주문관리

게시판(CS, 공지사항) : 작성, 수정, 삭제, 검색

담당 기능 : 상품관리

로그인 한 사용자의 지점 번호를 세션에서 받아와, 해당지점이 보유하고 있는 상품들의 목록을 불러온다.

상품 목록에서, 각 상품마다 모달을 사용해 상품에 대한 상세 정보를 조회 가능하게 구현 하였다.

정보 확인과 함께 즉시 수정할 사항이 있으면 수정 가능하도록 input 창을 함께 만들었다.

상세보기 시에 상품에 대한 이미지를 확인 할 수 있으며, 이미지 수정을 가능하게 만들었고, 적용 버튼을 누르면 ajax를 통해 바뀐 이미지를 바로 확인 할 수 있게 하였다.

삭제 버튼 클릭 시, 상품 삭제가 가능하다.

검색어를 통해 검색이 가능하며, 카테고리명, 상품명 등 여러 조건의 다중검색도 가능하다.
